安装 Java

点击下载适用于 x86 64 bit Windows 操作系统的 Zulu JDK & JFX 8. 注意 Minecraft 1.17 以及更新的版本不再支持 Java 8.

下载 Server Jar

Server Jar 是一个 Java 可执行文件, 以下运行它就可以运行一个服务端, 并生成需要的配置文件. 可以通过官方启动器, 官网或者第三方网站下载. 在官方启动器中可以通过依次点击: Minecraft: Java Edition, 配置, 新配置, (选择好版本, 在版本字段右上方有) 服务器下载按钮来下载 server. 或者可以搜索 Mojang 官网的新闻找到对应版本的 server. 最方便的是通过第三方网站 MCversions.net, 在 Stable Releases 中选择需要的版本, 点击 Download, 在打开的页面中点击 Download Server Jar.
